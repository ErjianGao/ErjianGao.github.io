Posted on 2020-07-02 Edited on 2021-10-09 In 软件工程 Views:

第八章

软件质量管理过程

  1. 软件质量计划：提出要求

    1. 实验设计
    2. 基准对照
    3. 质量成本分析
    4. 流程图方法
    5. 因果图方法

  2. 软件质量保证：通过评价项目整体绩效，建立对质量要求的信任

    质量保证活动——审计：是对过程或者产品的一次独立评估

    1. 项目执行过程审计
      1. 需求过程
      2. 设计过程
      3. 编码过程
    2. 项目产品审计

  3. 软件质量控制：确定项目结果与质量标准是否项目，同时，确定消除不符的原因和方法

    质量控制活动：

    1. 技术评审
    2. 代码走查
    3. 软件测试
    4. 返工

说出三种质量模型及其主要组成部分

  1. Boehm质量模型
    1. 软件的可使用性
    2. 软件的可移植性
    3. 可维护性
  2. McCall质量模型
    1. 产品修改
    2. 产品转移
    3. 产品运行
  3. ISO模型
    1. 质量特性
    2. 质量子特征
    3. 度量因子

质量保证与质量控制的关系：

质量控制QC：前期质量活动。是通过评价项目整体绩效,建立对质量要求的信任，提供项目和产品可视化的管理报告。这个任务本身并不能提高产品的质量，但是通过质量保证的一系列工作可以间接地提高产品的质量。质量保证一般由质量保证部门人员实施。

质量保证QA：后期质量活动。是确定项目结果与质量标准是否相符,同时,确定消除不符的原因和方法，它控制产品的质量，及时纠正缺陷。这个任务本身提高产品的质量，一般由开发人员实施。

质量保证是针对项目实施过程的管理手段，质量控制是针对项目产品的技术手段;实施质量保证是针对过程改进和审计的，强调的是过程改进和信心保证。实施质量控制是按照质量要求，检查具体可交付成果的质量，强调的是具体的可交付成果。

质量成本

  1. 预防成本
  2. 缺陷成本

第十二章

合同的类型

  1. 成本补偿合同

    1. 成本加成本百分比 CPPC

    2. 成本加固定费用 CPFF

    3. 成本加奖金 CPIF

      在成本的基础上加上利润加一定的奖励

  2. 固定价格合同

    1. 固定价格加奖励费 FPIF

      在CPIF的基础上固定最高价，如果大于目标成本则以最高价作为合同金额

    2. 固定总价 FFP

  3. 单价合同

    1. 一个产品或一个时间度量单位的价格

第十五章

挣值分析输入

输入 含义
BAC 总预算成本
TAC 计划完成时间
BCWS 计划完成工作的预算成本
ACWP 已完成工作的实际成本
BCWP 已完成工作的预算成本

挣值分析输出

输出 含义
SV BCWP - BCWS 进度差异
CV BCWP - ACWP 成本差异
SPI BCWP / BCWS 进度效能指标
CPI BCWP / ACWP 成本效能指标
EAC BAC / CPI 预测项目完成成本
SAC TAC / SPI 预测项目完成时间
VAC BAC - EAC 项目完成的成本差异
TCPI (BAC - BCWP) / (Goal - ACWP) 未完工指数

除了TCPI所有的输出都是越小越不好

如果TCPI大于1，则将来必须做的比计划好才能达到项目目标

BCWP的计算

  1. 50/50
  2. 1/100
  3. 经验加权
